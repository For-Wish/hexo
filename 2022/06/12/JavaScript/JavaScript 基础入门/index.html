JavaScript 基础

JavaScript 简介

JavaScript 是一门编程语言，可以实现很多的网页交互效果。

JavaScript 的组成

  • ECMAScript：规定了 js 基础语法核心知识
    • 比如：变量、分支语句、循环语句、对象等等
  • Web APIs：
    • DOM：操作文档，比如对页面元素进行移动、大小、添加删除等操作
    • BOM：操作浏览器，比如页面弹窗，检测窗口宽度、存储数据到浏览器等待

Web APIs 实现参考官方文档： MDN 网络文档 (mozilla.org)
